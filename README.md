# 📊 Student Marks Prediction

## 🎯 Project Overview

This project predicts student marks based on:

* Study Hours
* Attendance

## 🧠 Model Used

* Linear Regression

## 📈 Results

* R² Score: 0.99
* Accurate predictions

## 🔍 Key Insights

* Study hours strongly affect marks
* Attendance also plays an important role
* Sleep was removed due to low impact
